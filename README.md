# Getting started

### `Clone the repository`

```bash
git clone https://github.com/caio-andres/real-time-chat.git
```

### `Run back-end`

#### Open a terminal

```bash
cd chat-api
```

```bash
yarn install
```

```bash
nest start --watch
```

### `Run front-end`

#### Open a second terminal without close the first

```bash
cd chat-interface
```

```bash
yarn
```

```bash
yarn start
```

#### Use the URL `localhost:3000` in your browser
